# Climate Crisis

**Tagline:** "Navigate the Tempest, Survive the Change"

## Overview

Climate Crisis is an educational single-player game that raises awareness about climate change and its real-world impacts. Players are immersed in a world altered by climate-driven events and must navigate natural disasters, resource scarcity, and environmental degradation. The game combines exploration, decision-making, and narrative to inform and inspire action.

## Features

- Immersive environments — Explore diverse landscapes affected by climate change, such as arid deserts and degraded ecosystems.
- Compelling narrative — Make impactful decisions that shape your journey and highlight real-world climate challenges.
- Appealing visuals — Carefully selected assets and art direction create a comfortable and engaging experience.

## Installation

1. Clone this repository.
2. Open the project in the Godot editor.
3. Run the game from the Godot editor.

## Prerequisites

- Godot 4.x (the project was developed for Godot 4; using the latest stable 4.x release is recommended).
